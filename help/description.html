Chaque repertoire, modules et scripts sont decrit ci-dessous:

autoload

Chaque Script contenu dans ce repertoire est executé au démarrage d'une session PowerShell si le profile a été mis en place.

Nom Description
1-clipboard.ps1 Gestion du presse papier (Clipboard)
fonctions :
Get-Clipboard
Set-Clipboard
ccd : permet de faire faire un cd sur le chemin contenu dans le Clipboard
2-filter.ps1 filter(~fonctions) :
ConvertTo-KMG : convertir en KB, MB, GB ou PB
match : equivalent grep
exclude : equivalent grep -v
3-history.ps1 Inactif si la version de powershell est supérieur ou égale à 3.
Historique persistant + fonctions :
h : historique des 1024 dernières commandes
hg : Recherche dasn l'historique
hist : N'afficher que la propriété CommandLine de l'historique
4-import-module.ps1 Import des modules :
PSBookmark, PSRemoteRegistry, PSLogging, PSReadLine
fonctions :
importad : Import du module activedirectory
addvmware : import du module VMware PowerCLI
5-netfx.ps1 Raccourci .NET FrameWork.
fonctions :
now : équivalent Get-Date
ts : convertit un objet en chaîne de caractère
6-reload.ps1 Fonction LaunchConsole qui permet de lancer une nouvelle fenêtre PowerShell (avec kill de la fenêtre courante)
7-replace-prompt.ps1 Fonction prompt qui permet de personnaliser l'affichage du prompt PowerShell
Ajout de l'heure dans le prompt, affichage du repertoire cournat en couleur et limitation du nombre de caractère afficher.
8-scripts.ps1 Scripts de gestion des scripts/modules du psh-tools\autoload.
fonctions :
listps : lister les scripts psh-tools\autoload
editps : ouvrir dans Notepad++ les scripts psh-tools\autoload
exploreps : ouvrir \psh-tools dans l'explorer
loadps : charger dans la session powershell les scripts psh-tools\autoload
9-tools.ps1 Accès à certain outils.
fonctions :
tmpedit : Créer & edit le fichier C:\temp\tmp.txt, ajoute également la chaîne passée en "input"
explore : lance l'explorer dans le répertoire courant ou dans le chemin passé en paramètre
FixCRLF : Permet de corriger les CRLF (retour chariot) dans un fichier texte (txt,log,ini,etc...) passé en paramètre
10-exploit.ps1 Positionnement de plusieurs variables ainsi que quelques fonctions utiles pour l'écriture de script.
La liste des variables avec leurs valeurs & des fonctions dans la page Listing
11-fonctions.ps1 Fonctions utile à l'administration système.
La liste des fonctions est disponible sur la page Listing
12-aliases.ps1 Alias de commande ou de fonction
La liste des alias est disponible sur la page Listing
13-welcome.ps1 Message affiché lors du lancement d'une nouvelle session PowerShell.
Le message donne la date du jour, l'uptime de la machine et test si l'utilisateur est Admin
99-Psh-Help.ps1 Script d'aide & d'info sur le psh-tools (en cours de construction!)

bin

Nom Description
npp Notepad++ en version minimaliste (pas de theme, plugin, updater).
Office365 Installeur pour gérer Office365/Azure en PowerShell.
PortQry Utilitaire de test de port (TCP & UDP).
Sysinternals Les outils Sysinternals de M.Russinovitch.
7za.exe Utilitaire 7zip.
forfiles.exe Sélectionne et exécute une commande sur un fichier ou un ensemble de fichiers.
nvspbind.exe Utilitaire pour modifier le binding des cartes réseaux.
tail.exe Utilitaire permettant de faire de l'affichage en temps réel (pour les logs surtout).
wget.exe Utilitaire permettant de recevoir des fichiers depuis des sites web.

fic

Dossier permettant de stocker des fichiers utiles (par exemple : certificat, ini, flag boot, etc...)

lib

Dans ce dossier se trouve tous les scripts & modules qui peuvent servir à l'administration & l'exploitation.
Les modules sont placés dans Modules & les scripts dans Scripts.

Modules

La liste des modules & leur explication se trouve dans la page Listing.

Scripts

La liste des Scripts & leur explication se trouve dans la page Listing.

log

Repertoire dans lequel il faut déposer les logs.

Scripts

Répertoire contenant les scripts nécessaires à l'exploitation (purge de log, redémarrage de service).

Setup

Nom Description
GodMode Raccourci vers le GodMode : tous les items du panneau de configuration
TaskBar Contient un script qui mets quelques raccourcis dans la barre des tâches.
desactiver IPv6.reg Créer la valeur DisableComponents & assigne à 0x20 (Préférer IPv4 sur IPv6).
EULA_Sysinternals.cmd Mets les Accept EULA des sysinternals à Yes dans la base de registre.
Icone Network Desktop Documents Win2008 Créer des raccourci sur le bureau.
SetupPshProfile.ps1 & .cmd Créer le profile powerShell pour l'utilisateur courant.
View network connections.lnk Raccourci vers les connexions résaux.

posh-envt.ps1

Il s'agit du script à placer dans le profile PowerShell afin de charger l'environement psh-tools.
