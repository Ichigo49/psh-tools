Profile PowerShell

Le profil permet de définir des “préférences” à l’ouverture d’une session Powershell.

Par défaut, il n’y a pas de profil utilisateur Powershell. Il existe 4 types de profils différents sur Powershell :

  • %windir%\system32\WindowsPowerShell\v1.0\profile.ps1
    Ce profil s’applique à tous les utilisateurs et tous les environnements

  • %windir%\system32\WindowsPowerShell\v1.0\Microsoft.PowerShell_profile.ps1
    Ce profil s’applique à tous les utilisateurs, mais uniquement à l’environnement Microsoft.PowerShell

  • %UserProfile%\Mes documents\WindowsPowerShell\profile.ps1
    Ce profil s’applique uniquement à l’utilisateur actuel, mais affecte tous les environnements

  • %UserProfile%\Mes documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1
    Ce profil s’applique uniquement à l’utilisateur actuel et à l’environnement Microsoft.PowerShell

Afin de mettre en place, il suffit de créer ainsi qu'éditer le fichier profile pour executer le psh-tools :

New-Item $profile -ItemType file -force
Set-Content -Value ". <path-to-psh-tools>\posh-envt.ps1" -Path $profile
